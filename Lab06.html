For this lab, well be using the Challeng data set from the alr4 package. Further information on the data set can be found here.

  1. In the Challeng data set, the variable fail represents the number of O-rings that failed during prior flights. Use the ifelse() function (similar to the Lab 06 Example script) to create a new binary variable failed, where 0 will represent observations with no failed O-rings and 1 will represent observations with any (1 or more) failed O-rings.
data(Challeng, package = "alr4")

Challeng$failed <- ifelse(Challeng$fail >= 1, 1, 0)

  1. Create a second logistic regresison model that predicts failed with temp and pres. Use the anova() function to determine if this new model is an improvement (using a significance level of 5%) over the first model.
lr.challenge2 <- glm(failed ~ temp + pres, data = Challeng, family = "binomial")
anova(lr.challenge, lr.challenge2, test = "LRT")

The p-value is 0.2464. At a 5% level, the second model which used temp and pres as predictors is not a statistically significant improvement over the first model (which used just temp as a predictor).
