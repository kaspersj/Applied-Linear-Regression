Salaries of Professors

We left off Lab 3 working with the salary data set. Further information on the data set can be found here.

In Lab 3, we ran a regression model with a factor variable, comparing the salaries for different groups. Now we want to add in a quantitative predictor as well. With a quantitative predictor and a factor variable, we might be interested in determining if the groups have the same slope or if the relationship between X and Y differs by group.

Here, we want to know if the relationship between ‘years in current rank’ and ‘salary’ differ between the three ‘ranks’ (asst, assoc, and prof).

  1. So…should we leave all four predictor variables in the model or should we exclude some of them because of collinearity? Breifly describe a scenario in which each answer (leaving them all in and excluding some of them) could be the correct method.

If we are looking to find a model that provides the best overall fit, we would want to keep all of the variables in the model. The predictive power of a model is not reduced because of collinearity. If, on the other hand, we want to determine the individual effects of the predictors, we should think about removing the variables that are highly correlated with eachother because they can give us unreliable estimates.
